Conta & Sincronização

Conecte o Atalho do iPhone para importar dados do app Saúde automaticamente.

Credenciais para o Atalho iOS
URL Supabase
Anon Key
User ID (seu identificador fixo)
Como configurar o Atalho
1
Abra Atalhos → Novo Atalho. Nomeie "Serena Sync".
2
Adicione ações "Buscar Amostras de Saúde" para cada dado (período: Ontem):

Passos · Calorias Ativas · Tempo de Exercício · Horas em Pé
FC em Repouso · Variabilidade da FC · VO2 Máx · Saturação O₂
Análise do Sono · Peso Corporal · Frequência Respiratória
3
Adicione "Obter Conteúdo da URL":

URL: SUA_URL.supabase.co/rest/v1/health_sync
Método: POST
Cabeçalhos: apikey → Anon Key  ·  Content-Type → application/json
Prefer → resolution=merge-duplicates
4
Corpo JSON: inclua user_id (copie o User ID acima) + date (data de ontem) + todos os valores coletados.
5
Automação → Nova Automação Pessoal → Hora do Dia → 7h → todos os dias → selecione "Serena Sync" → desative "Perguntar antes de executar".
